# AI Math Tutor

An intelligent math practice system designed to help students master mathematics through personalized learning paths, adaptive difficulty, and a wide variety of problem types.

![Math Practice Demo](https://img.shields.io/badge/Status-Active-brightgreen) ![Python Version](https://img.shields.io/badge/Python-3.8%2B-blue)

## Features

- **Adaptive Learning**: Automatically adjusts difficulty based on performance
- **Comprehensive Coverage**: Covers all major Grade 7 math topics
- **Progress Tracking**: Monitors performance and identifies weak areas
- **Multiple Problem Types**:
  - Arithmetic operations
  - Word problems
  - Geometry
  - Data handling
  - And more!
- **Interactive CLI**: Easy-to-use command-line interface
- **Personalized Feedback**: Detailed explanations and progress reports

## Getting Started

### Prerequisites

- Python 3.8 or higher
- pip (Python package manager)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/iitkgpvivek/ai-math-tutor.git
   cd ai-math-tutor
   ```

2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

### Usage

1. Start the application:
   ```bash
   python enhanced_math_practice.py
   ```

2. Follow the on-screen instructions to:
   - Create a new student profile or load an existing one
   - Select topics to practice
   - View progress and statistics
   - Discover new problem types
